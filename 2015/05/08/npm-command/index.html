npm常用命令

1. npm install <name[@版本号]> 安装nodejs的依赖包

2. npm install <name[@版本号]> -g 将包安装到全局环境中
安装到全局在代码中，直接通过require()的方式是没有办法调用全局安装的包的。全局的安装是供命令行使用的。

3. npm install 读取当前目录下面 package.json 中的配置安装该项目的依赖包

4. npm install <name[@版本号]> —save 安装的同时，将信息写入 package.json 中
5. npm init 会引导你创建一个 package.json 文件，包括名称、版本、作者这些信息等
6. npm remove <name[@版本号]> 移除
7. npm update <name[@版本号]> 更新
8. npm ls 列出当前安装的了所有包
9. npm root 查看当前包的安装路径
10. npm root -g 查看全局的包的安装路径
11. npm help 帮助，如果要单独查看 install 命令的帮助，可以使用的 npm help install
注意：以上命令只是常用命令，并不是全部。
