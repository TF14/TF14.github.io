nodeJS安装以及故障解决

nodeJS安装

  1. 到nodeJS官网：https://nodejs.org/download/下载自己操作系统的.msi文件。
  2. 双击安装，路径自己定（为了方便下面说明我这里定为 E:/nodeJS ）中间有个选择add path的添加环境变量。
  3. cmd运行 node -v 可以看到版本号。

npm命令无法使用

npm 常用命令：点击这里
直接安装后可能npm命令无法使用，到 E:/nodeJS/ 下双击运行一下 node.exe 和 npm.cmd 再到 C:\Users(用户)\你的用户名\AppData\Roaming 下面看有没有npm的文件夹，没有就新建一个。
注意：AppData 为隐藏文件夹。

npm安装依赖包问题

因为npm默认安装包是在国外服务器下载（registry.npmjs.org），可能是网速问题。可以修改镜像源：

  1. 临时修改镜像源：
    npm install --registry 镜像源 (依赖包名称如：express，hexo等)
    如： npm install --registry http://registry.npm.taobao.org express
    这样只是临时的本次安装修改了镜像源
  2. 永久修改镜像源
    npm config set registry http://registry.npm.taobao.org
    这样以后安装就不用加镜像源设置了，默认为修改的镜像源。
