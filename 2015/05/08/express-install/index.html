express安装初始以及故障解决

express 安装

  1. 安装nodeJS（nodeJS安装）。
  2. npm express -g（npm 命令详情）。
  3. 如果安装遇到问题无法安装请查看（nodeJS故障解决）。

express 故障

express 命令无法使用
安装成功后注意一下cmd控制台 express 的版本型号，如果没注意可以 npm express -g 重新安装。或者键入 npm install express@版本号 -g 安装指定版本。express 4.X.X 键入 npm install -g express-generator，express 3.X.X 键入 npm install -g express-generator@3。然后重试就能使用了。

进入 http://localhost:3000 就能查看到你的第一个 express 项目了。
