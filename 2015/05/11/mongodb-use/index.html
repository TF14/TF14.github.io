MongoDB 学习笔记（三）：基本使用

成功启动 MongoDB （启动MongoDB）后，再重新打开一个命令行窗口输入 mongo ，就可以进行数据库的一些操作。

创建数据库

MongoDB 没有创建数据库的命令。
use 'test'命令切换数据库，当前不存在该数据库则创建该数据库。
注意：执行这句命令到下次用 use <db name> 切换数/‘创建’据库之前，下面所有用 db.fn 之类的操作都是针对该数据库。下面都是针对 test 数据库进行说明。

创建集合（表）

  1. 可以用 db.createCollection('user') 给 text 创建一个名为 user 的集合（表）。
  2. 用 db.user.save({name: 'aish',sex:'M'}) 来给 test数据库下面的集合user添加一条数据，如果该集合不存在会默认添加这个集合。

数据库基本命令

  1. show dbs 显示所有数据库列表
  2. use test (有test)切换/(无test)创建数据库
  3. db.dropDatabase() 删除当前使用数据库
  4. db.cloneDatabase(“ip地址”) 将指定机器上的数据库的数据克隆到当前数据库
  5. db.copyDatabase("A", "B", "ip地址") 将指定电脑上面A数据库中的类容复制到B数据库中
  6. db.getName() 查看当前使用数据库名称

集合基本命令

  1. show collections 显示当前使用数据库所有集合列表
  2. db.createCollection(<name>,[option]) 创建集合，name必填，option可无，可选参数有 {capped: <Boolean>, autoIndexId: <Boolean>, size: <number>, max <number>}。
