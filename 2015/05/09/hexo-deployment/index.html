新建github仓库

  • 点此登录github账号；如果还没有，点这里注册。
  • 点击右上角你名字旁边的加号 create new repository 创建一个新的仓库。
  • Repository name 填写 yourname.github.io（把 yourname 改成你的账号的名字，这里别乱填，不然不好找你的域名）。
